# Joshantbrown Tap

## How do I install these formulae?

`brew install joshantbrown/tap/<formula>`

Or `brew tap joshantbrown/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joshantbrown/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
